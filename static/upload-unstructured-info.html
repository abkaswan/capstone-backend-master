Uploading unstructured files?

An unstructured file just refers to any file that does not have any rows or columns. Vast majority of the files are unstructured. Performing data mining on unstructured is more challenging than structured.

This application accepts only 2 types of unstructured text files - TXT and DOCX.

What happens after I upload?

Make sure you press the Submit button after you upload.

You are provided with a choice on how you want to parse your data - tf-idf (term-frequency inverse document) or Latent Dirichlet allocation(LDA).

Get info about tf-idf at this link: https://en.wikipedia.org/wiki/Tf%E2%80%93idf (Right-click the link and select 'Open link in new tab').

Get info about LDA at this link: https://en.wikipedia.org/wiki/Latent_Dirichlet_allocation (Right-click the link and select 'Open link in new tab').

Basically tf-idf and LDA calculate specific values for each word for each file - all words are the columns and each file is a row. If a word is present in 2 files then it shall have 2 different values for each of the 2 files.

If you are uploading only 1 file, you have the option of each row representing a paragraph of the file instead of just one row representing one file. Select the 'Adding only 1 file' option as 'Yes'.

This type of tables are necessary to perform all the relevant data modelling operations on the uploaded files.

You can filter out words that you do not want by using word stemming and word lemmatization. They are provided as options and you can click on the respective help button for more info.
