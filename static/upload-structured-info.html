CSV file

CSV files are stored just like Excel files. Get info about CSV files at this link: https://en.wikipedia.org/wiki/Comma-separated_values (Right-click the link and select 'Open link in new tab').

A structured CSV file is one which has rows and where all columns are labelled. There shouldn't be any unlabelled column.

TXT file

A structured TXT is just like a structured CSV. The only difference each row is usually(not always) separated by a space(1 or more). Every row must be in a new line.

If each column is not separated by a space but by some symbol or character like comma(',') or semicolon';' they can be specified in the delimiter option below.

What happens after I upload?

Make sure you press the Submit button after you upload.

The file shall be uploaded and you can see the result in the 'Display Data' tab. You can perform any data cleaning, transformation, visualization or modelling on the uploaded dataset.
